 Coffee Shop Sales Analysis & Visualization 

 Project Description
This project provides a comprehensive analysis of coffee shop sales data. It demonstrates the full data lifecycle: from cleaning raw data using **Python** to building an interactive business dashboard in **Power BI**.

---

 Repository Contents
 **`python coffee shop.ipynb`**: The Jupyter Notebook containing the Python code for data cleaning and preprocessing.
 **`power BI coffee shop.pbix`**: The interactive Power BI dashboard file.
 **`code coffe shop.csv`**: The dataset used for the analysis.

---

 Technical Workflow
1.  **Data Cleaning (Python/Pandas):**  Removed duplicates and handled missing values.
    * Converted data types for dates and numerical values.
    * Prepared the final CSV for Power BI consumption.
2.  **Data Visualization (Power BI):**  Created KPI cards for total sales and orders.
    * Visualized sales trends by hour and day.
    * Identified top-performing product categories.

---

 Key Insights
* Optimized staffing levels by identifying peak sales hours.
* Highlighting the best-selling products to improve inventory management.
* Analyzing monthly growth to track business performance.
